**调用说明:**

1. **前置条件**：确保您的页面已正确引入私有部署提供的 paas.js 文件。  
2. **初始化**：在调用任何 load 函数之前，必须先调用 window.FH2.initConfig() 并传入正确的服务器地址、项目 ID 和组织密钥 (Project Token)。  
3. **加载**：调用相应的 load 函数，传入目标 div 的 ID 和该组件所需的特定参数（如航线 ID 或设备 SN）。  
4. **容器**：确保 HTML 中存在与 domId 参数匹配的 div 元素，作为组件渲染的目标。


这份表格总结了核心的调用方式和功能，您可以根据具体需求查阅 大疆司空2接口文档.md 获取更详细的接口参数和事件说明。
